Bakso Idle Clicker ==> pada dasarnya game clicker ini akan menghasilkan 1 bakso(uang) per 1 click.

Game Features:


| Button Utama |

1. Cursor (15) -- Berfungsi sebagai auto-clicker dimana saat cursor ini dibeli, maka bakso(uang) pun akan bertambah sendiri secara otomatis. 
2. Buka Cabang Bakso (100) -- Saat cabang baru dibeli, maka penghasilan bakso(uang) akan meningkat.

-- 

| Menu |

1. Mie & Bihun (1500)
2. Bakso Urat (5000)
3. Bakso Telur (10000)

-- Setiap menu baru tertentu dapat dibeli untuk meningkatkan pemasukan sehingga bakso(uang) yang didapatkan per detik semakin banyak.

| Alert/Warning |

-- Apabila uang tidak cukup untuk membeli upgrade tertentu, Alert/Warning akan pop up menyatakan bahwa bakso(uang) yang dimiliki kurang.
-- Ketika membeli menu baru, Alert/Warning akan pop up menyatakan "New Menu Unlocked".
